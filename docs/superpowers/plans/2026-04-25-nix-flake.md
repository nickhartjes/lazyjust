# Nix Flake Packaging Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `lazyjust` as a Nix flake providing `packages`, `apps`, `devShells`, `checks`, and an `overlays.default` for downstream NixOS / home-manager / nix-darwin consumers.

**Architecture:** A flake-utils-based flake that uses `crane` for granular Rust caching and pins the Rust toolchain via `rust-overlay`. Shared inputs (`commonArgs`, `cargoArtifacts`) live in `nix/common.nix`; package, checks, and devshell each consume them. The package wraps the binary with `just` on `PATH` so downstream users don't have to manage that runtime dep.

**Tech Stack:** Nix flakes, `flake-utils`, `crane`, `rust-overlay`, `makeWrapper`, `cargo` (1.78.0 toolchain).

**Spec:** `docs/superpowers/specs/2026-04-25-nix-flake-design.md`

**Implementer prerequisites:**
- Nix with flakes enabled (`nix --version` ≥ 2.18, `experimental-features = nix-command flakes` in `~/.config/nix/nix.conf` or `--extra-experimental-features` flag)
- macOS or Linux (verification runs on `aarch64-darwin` per spec; other platforms left to community)
- Network access for first-run input fetches

**Conventions used in this plan:**
- All file paths are relative to the repo root (`/Users/nick.hartjes/projects/nh/lazyjust-rs`).
- Run `nix` commands from the repo root unless stated.
- "Expected" lines describe what the command should print on success; partial matches OK.

## Task 4: Package derivation (`nix/package.nix`) and minimal `flake.nix` outputs

Wire the package together with the smallest viable outputs (`packages.default` only) so we can run `nix build` and prove the derivation works before layering more outputs on top.

**Files:**
- Create: `nix/package.nix`
- Modify: `flake.nix`

- [ ] **Step 1: Write `nix/package.nix`**

```nix
{ lib
, craneLib
, common
, just
}:

craneLib.buildPackage (common.commonArgs // {
  inherit (common) cargoArtifacts;

  pname = "lazyjust";

  postInstall = ''
    wrapProgram $out/bin/lazyjust \
      --prefix PATH : ${lib.makeBinPath [ just ]}
  '';

  meta = with lib; {
    description = "Terminal UI for the just command runner";
    homepage = "https://github.com/nickhartjes/lazyjust";
    license = with licenses; [ mit asl20 ];
    mainProgram = "lazyjust";
    platforms = platforms.unix;
  };
})
```

- [ ] **Step 2: Replace `flake.nix` with the per-system package wiring**

```nix
{
  description = "Terminal UI for the just command runner";

  inputs = {
    nixpkgs.url     = "github:NixOS/nixpkgs/nixos-unstable";
    flake-utils.url = "github:numtide/flake-utils";
    crane.url       = "github:ipetkov/crane";
    rust-overlay = {
      url = "github:oxalica/rust-overlay";
      inputs.nixpkgs.follows = "nixpkgs";
    };
  };

  outputs = { self, nixpkgs, flake-utils, crane, rust-overlay }:
    flake-utils.lib.eachDefaultSystem (system:
      let
        pkgs = import nixpkgs {
          inherit system;
          overlays = [ (import rust-overlay) ];
        };

        rustToolchain = pkgs.rust-bin.stable."1.78.0".default.override {
          extensions = [ "rust-src" "clippy" "rustfmt" ];
        };

        craneLib = (crane.mkLib pkgs).overrideToolchain rustToolchain;

        common  = pkgs.callPackage ./nix/common.nix  { inherit craneLib; };
        package = pkgs.callPackage ./nix/package.nix { inherit craneLib common; };
      in {
        packages.default  = package;
        packages.lazyjust = package;
      });
}
```

```nix
{
  description = "Terminal UI for the just command runner";

  inputs = {
    nixpkgs.url     = "github:NixOS/nixpkgs/nixos-unstable";
    flake-utils.url = "github:numtide/flake-utils";
    crane.url       = "github:ipetkov/crane";

    flake-utils.lib.eachDefaultSystem (system:
      let
        pkgs = import nixpkgs {
          inherit system;
          overlays = [ (import rust-overlay) ];
        };

        rustToolchain = pkgs.rust-bin.stable."1.78.0".default.override {
          extensions = [ "rust-src" "clippy" "rustfmt" ];
        };

        craneLib = (crane.mkLib pkgs).overrideToolchain rustToolchain;

        common   = pkgs.callPackage ./nix/common.nix   { inherit craneLib; };
        package  = pkgs.callPackage ./nix/package.nix  { inherit craneLib common; };
        checks   = import ./nix/checks.nix             { inherit craneLib common; };
        devShell = pkgs.callPackage ./nix/devshell.nix { inherit rustToolchain; };
      in {
        packages.default  = package;
        packages.lazyjust = package;
        apps.default      = flake-utils.lib.mkApp { drv = package; };
        devShells.default = devShell;
        checks            = checks // { inherit package; };

        let
          craneLib = (crane.mkLib final).overrideToolchain
            final.rust-bin.stable."1.78.0".default;
          common = final.callPackage ./nix/common.nix { inherit craneLib; };
        in {

            inherit craneLib common;
          };
        };
    };
}
```

## Risks and mitigations

- **Rust 1.78.0 missing from `rust-overlay`.** If `pkgs.rust-bin.stable."1.78.0"` evaluates to an error, fall back to `pkgs.rust-bin.stable.latest.default`. Document the fallback inline as a comment if used.
- **Interactive shell rc files override `PATH`.** The PTY spawns the user's `$SHELL` (`src/session/wrapper.rs:16`) and inherits the wrapper's `PATH`. A `.bashrc` that hard-resets `PATH` could mask `just`. Out of scope for this plan; users hitting this should install `just` themselves. The wrapper still helps the in-process `Command::new("just")` calls in `src/discovery/mod.rs:33,41`.
- **Darwin SDK frameworks insufficient.** If the Darwin build fails with a missing-symbol error from `arboard`, add `Foundation` to the framework list in `nix/common.nix`. The error message names the missing framework.
- **Closure size on Linux.** X11 + Wayland transitive closure adds a few MB. Acceptable per spec.

## Out of scope

- Submitting a derivation to `NixOS/nixpkgs`.
- Adding a `nix flake check` step to `.github/workflows/ci.yml`.
- Cachix / binary cache configuration.
- Static / musl builds.
